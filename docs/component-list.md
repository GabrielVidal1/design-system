# `@gabvdl/ui` — component list

Every component and hook in the library (v0.23.0), with a one-line description of what it is for. Skim this before building UI: if something here already covers the need, use it instead of writing a new component.

> Generated by `scripts/gen-component-list.mjs` (`npm run docs:list`) from the docs registry in `apps/docs/src/App.tsx` and the exports in `packages/ui/src/index.ts`. **Do not edit by hand** — add the description to the generator instead.

Live demos for every entry: [ui.gabvdl.xyz](https://ui.gabvdl.xyz) — 
the per-component page is `ui.gabvdl.xyz/#/c/<id>`.

```tsx
import { Button, Modal, useToast } from '@gabvdl/ui';
import '@gabvdl/ui/styles.css'; // or theme.css for tokens only
```
